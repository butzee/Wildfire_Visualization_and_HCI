Filtering

The application features three major methods to filter the data: Year, cause and size.

For the year selection, it is either possible to animate fires per day of a specific year, or to animate the cumulated fires for each year (Option "All Years").

For the cause selection, it is possible to select multiple causes at once, or all together. The same holds for the size selection.

Further configuration

The button with the speed icon, left from the Play button, can be used to set different animation speed.

The next button further left can be used to scale the wildfire size by different factors. This can come in handy when there are a lot of wildfires very small in size.

The button in the top right corner can be used to scale the map back to its original size.

If the type of card, on which the wildfires are rendered, is not to your liking, you can switch between 3 different card type at any point. Just click the button with the layer icon in the top right corner, below the previous one.

Lastly, the button with the circles icon can be used to switch between either a clustering of wildfire, which might be a better choice when there are a lot of wildfires in a small area, or a scatter, which might be a better choice when there are a lot of wildfires spread out.

Animation

After all filters have been set (or none, as this means all year, causes and sizes), the data first has to be loaded, before it can be animated.

This can be done by simply clicking the "Fetch Data" button in the top right. Depending on how unspecific the filters are set, the data loading might take a while.

After the data has been loaded, just click on the Play button next to the time slider.

One can also drag the timeslider or click on a specific point.
